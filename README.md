# 🏏 IPL Match Data Analysis

Analyzed 900+ IPL matches (2008–2024) using Python and Pandas.

 --What I did
- Explored the dataset using head(), info(), describe()
- Cleaned missing values and fixed inconsistent team names
- Used groupby and sorting to rank teams by total wins
- Visualized results using Matplotlib bar charts

--Tools Used
- Python, Pandas, Matplotlib
- Google Colab

--Dataset
IPL Complete Dataset from Kaggle (2008–2024)
